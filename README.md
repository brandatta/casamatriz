# Organizador Casa Matriz

App Streamlit para clasificar documentos, PDFs, imágenes y referencias editoriales de Casa Matriz en cinco categorías:

1. Identidad, marca y estrategia
2. Obra autoral, ensayos y borradores
3. Cursos, formación y archivo astrológico
4. Bibliografía, investigación y fuentes
5. Archivo visual, editorial y referencias

## Instalación local

```bash
python -m venv .venv
source .venv/bin/activate   # Mac/Linux
# .venv\Scripts\activate    # Windows

pip install -r requirements.txt
streamlit run app.py
